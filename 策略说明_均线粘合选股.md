# 创业板均线粘合选股策略说明（移动止盈版）

## 一、策略概述

本策略是一个基于均线粘合原理的量化选股策略，专门针对创业板股票（list_sector=2）设计。通过寻找短期均线（5日、10日、20日）粘合且位于中期均线（30日）上方的股票，捕捉股价即将启动的潜在机会。

**策略特点**：买入后持有，不做周期调仓，只做移动止盈止损。

---

## 二、核心逻辑

### 2.1 均线粘合原理

均线粘合是指多条不同周期的移动平均线在某一价格区间内相互靠近、纠缠在一起的现象。这通常意味着：
- 多空双方力量趋于平衡
- 市场处于盘整蓄势阶段
- 一旦突破，可能产生较大的趋势行情

### 2.2 选股条件

#### 条件1：均线粘合
计算三条短期均线的相互差异：
- `diff5_20` = |5MA - 20MA| / 20MA < 2%
- `diff10_20` = |10MA - 20MA| / 20MA < 2%
- `diff5_10` = |5MA - 10MA| / 10MA < 2%

**粘合度评分** = (diff5_20 + diff10_20 + diff5_10) / 3

评分越低，表示均线越粘合。

#### 条件2：位于30日均线上方
- 收盘价 > 30日均线
- 确保股票处于中期上升趋势中

#### 条件3：排序筛选
- 按粘合度评分升序排序
- 选择评分最低的股票（根据空仓位数决定买入数量）

## 三、交易规则

### 3.1 买入规则
- **买入时机**：有空仓位时，每日选股买入
- **选股范围**：创业板股票（list_sector=2）
- **买入价格**：开盘价
- **仓位分配**：等权分配，每只股票仓位 = 1 / 10（最大持仓10只）
- **避免重复**：排除已持仓的股票

### 3.2 卖出规则

#### 卖出条件1：移动止盈
- **触发条件**：盈利超过 10%
- **止损线上移**：止损线 = max(成本价, 10日均线)
- **执行卖出**：股价跌破止损线时卖出
- **目的**：保护已获得的利润，避免坐过山车

#### 卖出条件2：基础止损
- **触发条件**：收盘价跌破40日均线
- **目的**：防止趋势反转造成较大损失

### 3.3 交易流程
```
每日执行：
  ├── 检查持仓股票的止盈止损
  │     ├── 盈利≥10% → 启动移动止盈，止损线=MAX(成本价, 10日均线)
  │     ├── 已启动移动止盈且跌破止损线 → 卖出
  │     └── 跌破40日均线 → 卖出（基础止损）
  │
  └── 检查是否有空仓位
        └── 有则选股买入（最多持有10只）
```

## 六、策略优势

1. **趋势与盘整结合**：既要求均线粘合（盘整），又要求位于30日均线上方（趋势）
2. **创业板特性**：创业板股票波动性较大，均线粘合后的突破往往更有力度
3. **分散投资**：最多持有10只股票，降低单只股票风险
4. **纪律性交易**：机械化执行，避免人为情绪干扰
5. **移动止盈保护**：盈利10%后启动移动止盈，保护利润同时让盈利奔跑

---

## 七、风险提示

1. **市场风险**：策略无法预测系统性风险，大盘暴跌时可能失效
2. **参数敏感性**：均线粘合阈值（2%）、持仓数量和盈利保护比例会影响策略表现
3. **回测局限性**：历史回测不代表未来收益
4. **创业板风险**：创业板股票波动大，涨跌幅限制为20%
5. **移动止盈风险**：在震荡市中可能频繁触发止损

---

## 八、优化方向

1. **加入成交量过滤**：均线粘合时要求成交量萎缩，突破时放量
2. **行业分散**：限制同一行业的持仓比例
3. **动态止盈比例**：根据市场波动率调整盈利保护触发比例
4. **多因子结合**：结合RSI、MACD等技术指标
5. **仓位管理**：根据粘合度评分动态分配仓位

## 十一、参考指标

### 均线计算公式
```
MA_n = (Close_1 + Close_2 + ... + Close_n) / n
```

### 粘合度计算
```
diff5_20 = |MA5 - MA20| / MA20
diff10_20 = |MA10 - MA20| / MA20
diff5_10 = |MA5 - MA10| / MA10
convergence_score = (diff5_20 + diff10_20 + diff5_10) / 3
```

### 移动止盈计算
```
盈利比例 = (当前价格 - 成本价) / 成本价

if 盈利比例 >= 10%:
    止损线 = max(成本价, 10日均线)
    if 当前价格 < 止损线:
        卖出
```
