# 创业板均线粘合选股策略说明

## 一、策略概述

本策略是一个基于均线粘合原理的量化选股策略，专门针对创业板股票（list_sector=2）设计。通过寻找短期均线（5日、10日、20日）粘合且位于中期均线（30日）上方的股票，捕捉股价即将启动的潜在机会。

---

## 二、核心逻辑

### 2.1 均线粘合原理

均线粘合是指多条不同周期的移动平均线在某一价格区间内相互靠近、纠缠在一起的现象。这通常意味着：
- 多空双方力量趋于平衡
- 市场处于盘整蓄势阶段
- 一旦突破，可能产生较大的趋势行情

### 2.2 选股条件

#### 条件1：均线粘合
计算三条短期均线的相互差异：
- `diff5_20` = |5MA - 20MA| / 20MA < 2%
- `diff10_20` = |10MA - 20MA| / 20MA < 2%
- `diff5_10` = |5MA - 10MA| / 10MA < 2%

**粘合度评分** = (diff5_20 + diff10_20 + diff5_10) / 3

评分越低，表示均线越粘合。

#### 条件2：位于30日均线上方
- 收盘价 > 30日均线
- 确保股票处于中期上升趋势中

#### 条件3：排序筛选
- 按粘合度评分升序排序
- 选择评分最低的 top_n 只股票（默认10只）

## 三、交易规则

### 3.1 买入规则
- **调仓频率**：每5个交易日进行一次选股调仓
- **选股范围**：创业板股票（list_sector=2）
- **买入价格**：开盘价
- **仓位分配**：等权分配，每只股票仓位 = 1 / n

### 3.2 卖出规则
- **止损条件**：收盘价跌破40日均线
- **调仓卖出**：不在新一期选股列表中的股票
- **卖出价格**：开盘价

## 六、策略优势

1. **趋势与盘整结合**：既要求均线粘合（盘整），又要求位于30日均线上方（趋势）
2. **创业板特性**：创业板股票波动性较大，均线粘合后的突破往往更有力度
3. **分散投资**：最多持有10只股票，降低单只股票风险
4. **纪律性交易**：机械化执行，避免人为情绪干扰

---

## 七、风险提示

1. **市场风险**：策略无法预测系统性风险，大盘暴跌时可能失效
2. **参数敏感性**：均线粘合阈值（2%）和持仓数量会影响策略表现
3. **回测局限性**：历史回测不代表未来收益
4. **创业板风险**：创业板股票波动大，涨跌幅限制为20%

---

## 八、优化方向

1. **加入成交量过滤**：均线粘合时要求成交量萎缩，突破时放量
2. **行业分散**：限制同一行业的持仓比例
3. **动态调仓周期**：根据市场波动率调整调仓频率
4. **加入止损机制**：设置固定比例止损（如-8%）
5. **多因子结合**：结合RSI、MACD等技术指标

## 十一、参考指标

### 均线计算公式
```
MA_n = (Close_1 + Close_2 + ... + Close_n) / n
```

### 粘合度计算
```
diff5_20 = |MA5 - MA20| / MA20
diff10_20 = |MA10 - MA20| / MA20
diff5_10 = |MA5 - MA10| / MA10
convergence_score = (diff5_20 + diff10_20 + diff5_10) / 3
```
